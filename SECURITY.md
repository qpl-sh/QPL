# Security Policy

If you discover a vulnerability in QPL, thank you and please report it privately so we can fix it before disclosure.

Preferred contact methods:

- Create a private security advisory on GitHub (recommended): https://docs.github.com/en/code-security/security-advisories
- Or email security@qpl-sh.dev (replace with the appropriate address)

Please include:
- A short description of the issue and impact
- Steps to reproduce (minimum example if possible)
- Suggested mitigation (optional)

We aim to respond within 48 hours for critical issues.
